# Store / Release Checklist

## Product & Compliance
- [ ] Final app name and subtitle approved
- [ ] Privacy policy URL published
- [ ] Support email confirmed
- [ ] Terms/usage text reviewed

## Assets
- [ ] App icon (1024x1024 source)
- [ ] Screenshot set for phone + tablet
- [ ] Short and long description copy
- [ ] Feature graphic / promo image

## Technical Validation
- [ ] `npm run typecheck` passes
- [ ] `npm test -- --run` passes
- [ ] `npm run build` passes
- [ ] PWA installability check passes
- [ ] Offline camera intake smoke test passes

## Operational Readiness
- [ ] Rollback plan documented
- [ ] Release notes prepared
- [ ] Monitoring/alerts configured
- [ ] Post-release verification checklist prepared
